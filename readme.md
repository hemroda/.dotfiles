.dotfiles
===


## Install xcode
`xcode-select --install` and `sudo xcodebuild -license`  


## Setup git
`chmod +x git_setup.sh`  
`./git_setup.sh` 


## Symlink files to appropriate places
`make`


## Install homebrew
`chmod +x homebrew.sh`  
`./homebrew.sh`


## Install all apps using brew
brew bundle
